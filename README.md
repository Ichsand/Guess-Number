# Guess-Number

https://roadmap.sh/projects/number-guessing-game

1. Choose difficulty by entering number between 1, 2, and 3 depends of their levels.

2. Guess by entering number in between 1 and 100. Watch carefully for attempts.

3. You can also type "hint" or "exact" to easy guessing

4. If guessing correctly, the score will appears depends on how many attempts to guess. The lower you guess the higher score you get
